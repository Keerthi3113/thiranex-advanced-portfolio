# Personal Portfolio Website

## Overview

A fully responsive personal portfolio website developed using Semantic HTML5, Advanced CSS3, Flexbox, CSS Grid, and JavaScript.

## Features

* Semantic HTML5 structure
* Accessibility support (ARIA labels, skip links)
* Responsive mobile-first design
* CSS Grid layouts
* Flexbox alignment
* Dark/Light theme toggle
* Glassmorphism effects
* SEO-friendly metadata
* Multi-page navigation

## Technologies Used

* HTML5
* CSS3
* JavaScript
* CSS Grid
* Flexbox

## Pages

* Home
* About
* Projects
* Contact
